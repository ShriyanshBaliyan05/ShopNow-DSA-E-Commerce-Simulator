📊 Performance Analysis (Big-O Time Complexity)

Task Data Structure Algorithm Time Complexity Why?
Search by Product ID Hash Table Hashing O(1) Direct index access via hash function
Search by Name Array Linear Search O(n) Checks each item one by one
Sorted Search (by price) Array Binary Search O(log n) Divide & conquer on sorted data
Sort Products Array Quick Sort O(n log n) Efficient for large datasets (avg case)
