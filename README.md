# 🛒 ShopNow – DSA E-Commerce Simulator

## 📌 Overview

ShopNow is a Data Structures and Algorithms (DSA) based E-Commerce Simulator developed in C. The project demonstrates how various data structures can be used to model and manage core functionalities of an online shopping platform.

Unlike a traditional e-commerce website, the primary goal of this project is to showcase the practical application of DSA concepts in a real-world scenario.

## 🎯 Project Objectives

* Demonstrate the use of Data Structures in an e-commerce environment.
* Simulate operations such as product management, user interactions, and order processing.
* Provide a clear understanding of how DSA concepts are applied in software systems.
* Bridge the gap between theoretical DSA knowledge and practical implementation.

## 🛠️ Technologies Used

* C Programming
* Data Structures & Algorithms
* HTML (for visualization and explanation)

## 🧩 Data Structures Used

The C implementation demonstrates the use of various data structures, including:

* Arrays
* Linked Lists
* Stacks
* Queues
* Searching Algorithms
* Sorting Algorithms

These structures are integrated to simulate different components of an e-commerce platform such as product catalog management, user operations, and order handling.

## 🌐 HTML Implementation

The HTML file is not intended to run the e-commerce system itself.

Instead, it serves as a visual representation and documentation of the project by:

* Explaining the overall system architecture.
* Showing where different data structures are used.
* Demonstrating how DSA concepts map to e-commerce functionalities.
* Providing an easier way to understand the project workflow.

## ▶️ Running the Project

Running the C file is optional for understanding the project.

The repository can be explored through:

1. The C source code, which contains the actual DSA implementation.
2. The HTML file, which explains how and where the data structures are utilized throughout the system.
